## Upgrading to the Latest Version

Follow these steps to ensure smooth updates in your Intelera projects.

<Steps>
  <Step title="Backup Projects" icon="database">

    Export your sites from the dashboard.

````bash
# From Intelera CLI (install via npm)
npm install -g @intelera/cli
intelera export --project my-site
````

  </Step>
  <Step title="Update Application" icon="download">

    Refresh your browser or reinstall the desktop app.

    <Tabs>
      <Tab title="Web" icon="globe">
        Clear cache and reload at `https://app.intelera.ai`.
      </Tab>
      <Tab title="Desktop" icon="monitor">
````bash
# macOS with Mobirise
brew upgrade intelera-mobirise
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Test Changes" icon="check-circle">

    Preview a template from the new library and deploy a test site.

  </Step>
</Steps>
